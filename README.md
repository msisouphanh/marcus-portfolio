## E-portfolio

A personal e-portfolio website designed to showcase my projects, skills, and experience.

### Technologies
- `React`
- `TypesScript`
- `HTML`
- `Tailwind CSS`
- `Vite`

## Running the Portfolio
1. Clone the repository to your local machine.
2. In your command prompt, install Vite by running: `npm install vite@latest`
3. Open a terminal in your code environment, in your terminal `npm install` to install the required dependencies.
4. Start the development server using: `npm run dev`
5. Open your browser and navigate to `http://localhost:5173`
